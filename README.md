# project-test

## Get started 

- install python 3.7.0
- pip install -r requirements.txt
- python -m venv /path/to/new/virtual/environment
- path_to_virtualenv/Scripts/activate
- rasa run activate
- rasa train
- rasa shell

## Run in browser

- path_to_virtualenv/Scripts/activate
- rasa run actions
- rasa run -m modals --enable-api --cors "*"
- open index.html in browser
